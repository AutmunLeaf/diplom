# Браузерное приложение для заполнения форм КС-2 и КС-3

## Описание

Это Django-приложение для удобного заполнения строительных актов по форме КС-2. 
Приложение разработано с учетом требований организации [АО «МОЭнергоСервис»](https://www.mo69.ru/).

## Возможности

### Форма КС-2
- ✅ Создание и редактирование форм КС-2
- ✅ Ввод данных в ручную (инвестор, заказчик, подрядчик, объект)
- ✅ Выбор единиц измерения из списка (м², м³, м, шт, точка и др.)
- ✅ **Ограничение на количество работ - не более 15 позиций**
- ✅ Автоматический расчет стоимости по каждой позиции
- ✅ Подсчет итоговой суммы
- ✅ Возможность удаления работ
- ✅ Заполнение шапки формы (данные договора, отчетный период, подписи)

### Форма КС-3 (заготовка)
- 📌 Зарезервировано место для будущего шаблона КС-3
- 📌 Вы сможете загрузить свой шаблон позже

## Установка и запуск

### Требования
- Python 3.8+
- Django 4.0+

### Установка зависимостей
```bash
pip install django openpyxl reportlab
```

### Миграции базы данных
```bash
python manage.py makemigrations
python manage.py migrate
```

### Запуск сервера
```bash
python manage.py runserver 0.0.0.0:8000
```

После запуска откройте в браузере: http://localhost:8000

## Модели данных

### KS2Form
Основная модель для хранения данных формы КС-2:
- Данные инвестора, заказчика, подрядчика
- Данные о строительстве и объекте
- Номер и дата договора
- Отчетный период
- Подписи сторон

### WorkItem
Модель для хранения позиций работ:
- Номер позиции
- Наименование работы
- Единица измерения (выбор из списка)
- Количество
- Цена за единицу
- Автоматический расчет общей стоимости

### KS3Form
Заготовка для будущей формы КС-3 (справка о стоимости).

## Ограничения

- **Максимальное количество работ в одной форме: 15**
- При достижении лимита добавление новых работ блокируется
- Для добавления новых работ необходимо удалить существующие

## Админ-панель

Для доступа к админ-панели создайте суперпользователя:
```bash
python manage.py createsuperuser
```

Затем перейдите по адресу: http://localhost:8000/admin/

## Дальнейшее развитие

Для добавления поддержки КС-3:
1. Загрузите шаблон КС-3 в папку `media/templates/`
2. Расширьте модель `KS3Form` необходимыми полями
3. Добавьте формы и представления для КС-3
4. Реализуйте экспорт в PDF/Excel аналогично КС-2
